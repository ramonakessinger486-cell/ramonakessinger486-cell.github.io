量化启动阶段

这是一个非常典型的**“冷启动”**问题。

作为一个大一新生，你没有足够硬的简历去通过大厂（幻方/九坤）的HR筛选，但你又急需实战经验来填充你的简历和大脑。

“不要钱”是一个很好的筹码，但你需要找对“交易对手”。
对于量化领域，大厂不缺免费劳动力（甚至怕泄密），缺人的是中小型私募和学术导师。

我为你规划了三条**“短平快”的切入路径，让你在大一暑假就能拥有“真实的项目经历”**：

路径一：国内量化平台的“云实习” (The Sandbox Mode)

—— 门槛最低，反馈最快，完全自主

如果你找不到线下的公司，那就把互联网平台当成你的实习公司。

目标平台： 聚宽 (JoinQuant)、米筐 (RiceQuant)、优矿 (Uqer)。

这些是国内最大的量化回测平台，自带数据（A股历史行情）、自带环境（Python）、自带社区。

如何操作（伪装成实习）：

注册账号： 免费。

找策略： 去社区里找“热门策略”（比如：小市值轮动、双均线策略、多因子策略）。

复现 (Replication)： 不要只看，自己一行一行把代码敲进去，跑通回测。

改进 (Optimization)： 试着改几个参数，或者换个时间段，看看收益率怎么变。

产出（写在简历上）：

项目名称： 基于多因子模型的A股量化择时策略研究。

角色： 独立研究员。

内容： 使用 Python 和 JoinQuant 平台，清洗了 2018-2024 年的沪深300数据，构建了动量与反转因子，实现了年化 15% 的回测收益。

评价： 这是最适合大一新生的“自制实习”。HSG 的教授看重的是你“做了什么”，而不是你在哪里打卡。

路径二：“冷邮件”狙击中小型私募 (Cold Emailing Boutiques)

—— 成功率低，但一旦成功就是从0到1

北京和上海有大量管理规模在 5-20亿 的中小型量化私募。他们缺人，且没有大厂那么繁琐的流程。

寻找目标：

去 “私募排排网”。

筛选：北京/量化/股票策略。

你会看到几百家公司的名字。

寻找联系方式：

去他们的官网、公众号，或者由 中财校友群 打听。

往往能找到 hr@… 或者 info@… 的邮箱。

你的“销售话术”（Cover Letter）：

标题： 【免费实习申请】中财金工大一 | Python/SQL熟练 | 数学满分 | 可全职3个月

正文核心： “我不要工资，不留用。我刚学了 Python 和 SQL，我可以帮团队做数据清洗 (Data Cleaning)、爬虫 (Crawling) 和 文档整理。我只想在一个专业的环境里学习。”

成功率： 发 50 封邮件，可能有 2-3 个回复。这就够了。 只要有一家让你去，你就赢了。

路径三：成为中财老师的“数据民工” (Faculty RA)

—— 最稳妥，且能拿到推荐信

你学校里那些 清北复交出身的年轻讲师，他们面临着巨大的**“非升即走”**的科研压力。他们非常缺人帮他们处理脏数据。

操作：

打开学院官网，找那些**“讲师”或“副教授”**（不要找大牌教授，他们不缺人）。

看他们的研究方向：如果是 Empirical Asset Pricing（实证资产定价） 或 FinTech，那就是你的目标。

直接去办公室堵门 或者 发邮件。

话术： “老师，我看过您的论文（哪怕只看了标题）。我是大一金工的，自学了 Python 和 Stata，暑假留校，想帮您处理数据，打打下手，不要钱。”

收益：

接触到最前沿的学术数据（CSMAR/Wind）。

如果干得好，大三申请时，他的一封推荐信就是你的通行证。

INTJ 的“避坑”指南

拒绝“付费内推” (Pay-to-Work)：

市面上有很多中介说“交2万块，保录取中金/券商远程实习”。

全是假的 (Fake)。 这种实习不走人事系统，不仅没用，而且是对你智商的侮辱。INTJ 绝不交这种智商税。

做好当“清洁工”的准备：

大一去实习，99% 的工作是洗数据（Data Cleaning）。

不要嫌烦。数据清洗是量化的基本功。 你能把脏数据洗干净，比你会背高深的公式更有价值。

现阶段最佳执行方案

我建议你采用 “路径一（平台自研）” + “路径三（找老师）” 的混合策略。

寒假在家（西安）： 主攻 JoinQuant/RiceQuant。自己写出 2-3 个策略代码，保存好。这是你的**“作品集”**。

大一下学期（开学后）： 拿着你的 Python 代码和成绩单，去找学院里的年轻老师，预定暑假的 RA（研究助理） 位置。

这样，你大一暑假就不用像没头苍蝇一样乱撞了。
你在给未来的简历造血。

行动起来，未来的 Quant。
