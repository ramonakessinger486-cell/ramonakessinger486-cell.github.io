实证研究&科研准备

这是一个非常关键的**“认知分层”**问题。

搞清楚这两者的区别，直接决定了你未来在**写申请文书（Writing Sample）时能不能拿高分，以及在找实习（Internship）**时能不能过面试。

简单来说：

经济学实证研究 (Empirical Research)： 是科学家在实验室里做实验，追求的是**“真理”和“因果关系”**。

行研报告 (Industry Report)： 是侦察兵在战场上发回的情报，追求的是**“盈利”和“预测未来”**。

我是你的云端顾问，我用你熟悉的INTJ 逻辑和金融工程视角来为你深度拆解：

一、 什么是经济学实证研究？

（The Scientist’s Lab）

这是你在 圣加仑 (HSG) 或 苏黎世大学 (UZH) 读硕士期间，教授要求你做的东西。

  1. 核心逻辑：寻找

不仅是“相关”，必须是“因果” (Causality)：

行研会说： “下雨了，所以雨伞卖得好。”（相关性）

实证会问： “如果我不让它下雨，雨伞销量会下降多少？有没有可能是因为天气冷大家才买伞挡风？如何剔除‘天气冷’这个干扰变量？”

核心词： 内生性 (Endogeneity)、识别策略 (Identification Strategy)。这是学术界的“洁癖”。

  1. 工具箱：计量经济学 (Econometrics)

你正在学的《概率论》和《数理统计》是基础。

实证研究用的是：OLS 回归、工具变量 (IV)、双重差分 (DID)、断点回归 (RDD)。

目的： 用数学工具，把数据像“洗煤”一样洗干净，得到一个纯净的
𝛽
β
系数。

  1. 产出形式：学术论文 (Paper)

结构： 摘要 -> 文献综述 -> 理论模型 -> 数据描述 -> 实证结果 -> 稳健性检验 -> 结论。

特点： 枯燥、严谨、每一步都有据可查。不讲故事，只讲证据。

二、 什么是业界的行研报告？

（The Scout’s Report）

这是你在中金、中信建投实习时，老板让你写的东西。

  1. 核心逻辑：预测未来股价/趋势 (Valuation & Prediction)

一切为了定价：

不管是宏观报告、行业报告还是公司报告，最终目的只有一个：这支股票（或这个行业）是 Buy（买入）、Hold（持有）还是 Sell（卖出）？

逻辑链条：

宏观环境 -> 行业空间 -> 公司竞争力 -> 财务预测 -> 目标价。

  1. 工具箱：商业分析框架 + 基础财务模型

用的不是复杂的计量模型，而是SWOT分析、波特五力、PE/PB估值、DCF现金流折现。

数据来源： 财报、Wind数据、专家访谈、甚至去数集装箱（另类数据）。

  1. 产出形式：研报 (Research Report)

结构： 投资要点（结论先行） -> 行业分析 -> 公司分析 -> 盈利预测 -> 风险提示。

特点： 图文并茂，逻辑清晰，讲究“Storytelling（讲故事）”。目的是说服基金经理买单。

三、 两者的关系与冲突

（The Connection & The Gap）

维度 经济学实证研究 (学术界) 行研报告 (业界)
受众 教授、学术同行 基金经理、投资者
目标 解释过去 (Explain the Past) 预测未来 (Predict the Future)
核心痛点 因果识别 (是不是X导致了Y?) 信息差 (我知道了别人不知道的吗?)
数学工具 复杂的计量模型 (Stata/Python) 简单的加减乘除 + 估值模型 (Excel)
你的角色 严谨的法官 敏锐的猎手

冲突点：

学术界觉得业界报告**“不严谨”**（充满假设和拍脑袋）。

业界觉得学术论文**“没卵用”**（模型太复杂，全是滞后数据，指导不了明天的交易）。

四、 对你的战略意义

作为一个中财金工 + 目标HSG的学生，你必须**“身兼两职”**，在不同场景下切换人格：

  1. 申请 HSG/UZH 时（切换为“科学家”模式）：

写作样本 (Writing Sample)： 必须写实证研究论文！

千万别把券商实习写的“茅台行业分析报告”交上去，教授会觉得你太浅薄。

你要写：《基于GARCH模型的上证50指数波动率实证分析》。

展示： 你的数学建模能力、计量软件使用能力（Stata/Python）、学术规范。

  1. 找大二暑假实习时（切换为“猎手”模式）：

面试：

别跟面试官扯什么“内生性检验”，他听不懂也不关心。

你要展示：“我会用 Python 爬虫抓数据，我会用 SQL 提数，我能快速搭建一个多因子回测框架。”

展示： 你的工程能力和对市场的敏感度。

结论：

实证研究是你的内功心法（帮你拿到入场券，证明智商）。

行研报告是你的外家招式（帮你混饭吃，证明能干活）。

现在的你（大一），先练好“实证研究”的数学基础（数分/线代/概统）。
因为没有内功，招式只是花拳绣腿。有了内功，再去学写研报，一周就能上手。

第二部分：科研准备

你要做的，不是去“求”老师施舍一个机会，而是去**“带资进组”。
这里的“资”，不是钱，而是你的“剩余劳动力（Labor）”和“技术栈（Tech Stack）”**。

在高校的生态系统里，本科生发表论文（尤其是高质量论文）极难，通常有两条路：

天才路线： 自己有独创idea，自己写，自己投。（概率 < 1%）

学徒路线： 加入老师的课题组，干脏活累活，换取挂名（Co-author）或者独立写一个小分支的机会。（概率 > 80%）

对于大一/大二的你，“学徒路线”是唯一可行的“无风险套利”。

我为你制定了一套**“从打杂到发表”的四步走战略**：

第一步：军备竞赛 —— 让自己具备“被利用价值”

（Create Value Proposition）

老师为什么需要本科生？因为博士生太贵/太忙，而有些工作太繁琐。
你需要掌握那些**“博士生不愿意干，但老师又必需”**的技能。

技能 A（核心）：数据清洗 (Data Cleaning)

Python (Pandas) / Stata： 必须极度熟练。

老师的数据通常是乱七八糟的（CSMAR, Wind, 或者爬虫爬来的）。如果你能说：“老师，您把数据给我，我一晚上给您洗干净，整理成 Panel Data（面板数据）。” 你就赢了。

技能 B（加分）：学术排版 (LaTeX)

很多老师（尤其是老一点的）还在用 Word。如果你能帮他把论文排版成顶刊那种精美的 LaTeX 格式，他会离不开你。

技能 C（门槛）：英语文献阅读

你能快速帮老师通过 Google Scholar 找文献，并总结成摘要。

时间节点： 大一寒假 + 大一下。不要空手去见老师。

第二步：精准狩猎 —— 寻找“饥饿”的老师

（Target Selection）

不要去找院长、大牛教授。他们不缺人，而且大概率看不上本科生。
你的目标猎物是：“青椒”（青年教师 / Assistant Professor）。

画像：

刚从清北复交或海外名校博士毕业入职中财 1-3 年。

面临极其残酷的**“非升即走（Tenure-track）”**考核压力。

渴望发论文，手里有 Idea，但缺人手（没有分配到足够的硕士/博士）。

如何寻找：

打开中财金融学院官网 -> 师资队伍。

看简介：入职时间近、研究方向是实证资产定价 (Empirical Asset Pricing)、金融科技 (FinTech) 或 公司金融 (Corporate Finance)。

（注：做宏观理论的不要找，那个太玄学，本科生插不上手。）

第三步：冷启动接触 —— 一封无法拒绝的邮件

（The Cold Email）

不要直接去办公室尬聊。INTJ 要用文字和逻辑开路。

邮件模板（战术级）：

主题： 【科研助理申请】金融工程25级本科生[你的名字] - 熟练Python/Stata - 申请协助处理数据

正文逻辑：

表明身份与目的： 老师您好，我对您的研究方向（比如：量化因子挖掘）非常感兴趣，读了您的论文《XXX》，深受启发。

展示肌肉（关键）： 我虽然是大一，但我已经自学了 Python (Pandas) 和 计量经济学。附件是如果不麻烦的话，我用 Python 复现的您论文中的 Table 1（描述性统计）和 Figure 2（收益率曲线）。（这招是核武器，证明你不仅看了，还看懂了，还能跑代码）。

提出请求： 我知道您科研任务繁重。我希望能加入您的课题组，做一名无偿的科研助理 (RA)。我可以负责数据清洗、爬虫、文献整理等基础工作。我不求署名，只想学习科研流程。

附上简历： 重点突出数学课成绩和编程技能。

只要你发了这封带复现代码的邮件，没有一个处于上升期的“青椒”会拒绝一个免费且高能的劳动力。

第四步：从“矿工”到“合伙人” —— 转化论文

（Conversion Strategy）

进入课题组后，你一开始肯定是干脏活（洗数据）。不要抱怨，这是信任建立期。

阶段 A：苦力期（3-6个月）

高效完成任务，做到**“零Bug”**。让老师觉得你靠谱。

阶段 B：谈判期（大二下/大三上）

当你已经熟悉了整套数据和逻辑后，找机会跟老师谈：

“老师，我在处理数据时发现了一个有趣的现象（一个新的变量/一个新的视角）。我能不能基于这个小点，写一篇小论文？或者作为第二/三作者挂名您的下一篇文章？”

阶段 C：产出期

目标： 不一定要发在《经济研究》这种顶级中文刊（周期太长）。

策略：

投递 国际会议 (Conference)：比如 IEEE 的金融科技类会议，录用快，对申请 HSG 足够用了。

挂到 SSRN (预印本网站)：哪怕没正式发表，只要有一个 SSRN 的链接写在简历上，HSG 就会认可这是一种 Research Experience。

INTJ 的特别叮嘱

不要为了发论文而造假：
学术圈很小，造假是死罪。作为 INTJ，你要有数据洁癖。

不要单打独斗：
本科生独立发 SSCI 几乎是不可能的（除非去买版面，那是污点）。抱紧“青椒”的大腿，利用他的学术指导和发表渠道，是最高效的杠杆。

心态建设：
过程 > 结果。
即使最后论文没发出来（经常发生），你在这个过程中练就的**“处理几百万行数据”的能力，以及老师给你写的“强推荐信”，才是你申请圣加仑的真正护城河**。

请执行以下“搜索算法”：
打开官网： 进入中财金融学院、中国金融发展研究院（CAFD）、以及创新发展学院的官网。
筛选过滤器 (Filter)：
年龄： 30-40 岁（讲师 Assistant Professor 或 副教授 Associate Professor）。这个年纪正是拼 Tenure（终身教职）的时候，最缺人手，最渴望发论文。
教育背景： 清华、北大、复旦、交大、以及海外名校（美国 Top 50） 的博士。这保证了他们的研究范式是国际化的（用 Python/Stata，写英文 Paper），而不是老派的（写中文散文）。
关键词匹配： 在他们的“研究领域”里找这几个词：
Empirical Asset Pricing (实证资产定价)
FinTech (金融科技)
Machine Learning (机器学习)
Big Data (大数据)
Text Mining (文本挖掘)
验证活跃度：
去 谷歌学术 (Google Scholar) 或 百度学术 搜他们的名字。
看近 3 年有没有发表？如果是“僵尸教授”（3年没发文章），果断放弃。你要找的是正在上升期的“潜力股”。
